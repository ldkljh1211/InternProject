# 2. 그룹 프로젝트
공간정보 산업 개선사항 발굴 프로젝트

## 기획 배경
* 인턴(9명) 주도로, 국토지리정보원 업무와 관련된 공간정보 산업의 문제점 발굴 및 개선방안 제안을 통한 효율적인 업무 도모

## 기간
* 24.11.15 ~ 24.12.30

## 주요 내용
* 내부 설문조사를 통한 문제점 발굴
* 개선안 도출 (인턴 브레인스토밍)
* 내부 전문가 피드백
* 최종안 확정 및 발표

## 나의 주요 역할
* Python을 이용한 데이터 시각화
* 직원들의 설문 내용을 바탕으로 형태소 분석, 키워드 추출 및 빈도 계산, 불용어 제거
* 머신러닝 기반 감정 분석 모델 제작 (자연어 처리(NLP), SVM 모델, 정확도: 97%)
